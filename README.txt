Final minimal upload package

Zenodo archived release: v1.1.0, https://doi.org/10.5281/zenodo.21311023

Use these files for submission preparation:
- ieee_access_manuscript_final_submission.docx: main manuscript.
- command_layer_relay_tradeoff_public_repo_v1.zip: public reproducibility repository for Zenodo/supplementary upload.
- figure_source_pngs.zip: source PNG figures if requested by the submission system.
- ieee_access_manuscript_final_submission_check.pdf: local render check only; upload only if the portal requests a PDF.
- CITATION.cff: visible citation metadata for the manuscript/repository package.
- manuscript_references.txt: reference list extracted from the manuscript.
- repo_file_manifest.txt: visible summary of the nested public repository zip contents.

Internal audit reports are intentionally excluded from this minimal package.
